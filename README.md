# React boilerplate

This is a react boilerplate template using tailwind.

## Install

```bash
  git clone https://github.com/missboa/react-boilerplate.git my-app
```

Installing npm

```bash
  cd my-app
  npm install
```

Removing git

```bash
  rm -rf .git && rm -rf README.md
```

Running the project

```bash
  npm run dev
```
